DASA & CIWG 2025 College Predictor (static webpage)

Files:
- index.html  (open in a browser via a local web server)
- styles.css
- app.js
- data.json   (program dataset with overall opening/closing ranks)

How to run locally:
1) Put all files in a folder.
2) Start a local server from that folder:
   - Python:  python -m http.server 8000
3) Open in browser:
   http://localhost:8000

Why a local server?
Browsers often block fetch() from local files. Running a tiny server avoids that.
